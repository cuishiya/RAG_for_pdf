📋 项目简介

这是一个基于 FAISS 和 LangChain 的智能问答系统，专门用于查询崔世亚的个人信息。系统会自动处理知识库中的多个PDF文档，构建向量数据库，并支持自然语言问答，可迁移为任何人的个人知识库。




⚙️ 环境要求

1️⃣配置必需的Python包

2️⃣需要拥有阿里云DashScope API密钥并设置为系统环境：

Windows

set DASHSCOPE_API_KEY=your_api_key_here

Linux/Mac

export DASHSCOPE_API_KEY=your_api_key_here




✨ 功能特点

✅ 多PDF文件处理：自动读取并处理知识库文档中的所有PDF文件

✅ 智能向量化：使用阿里云DashScope的文本嵌入模型进行向量化

✅ FAISS向量检索：高效的相似度搜索

✅ 来源追踪：精确显示答案来源的文件名和页码

✅ 智能缓存：首次运行后保存向量数据库，后续直接加载，节省时间

✅ 大模型问答：使用通义千问DeepSeek-V3模型生成答案




🚀 使用方法

1️⃣ 首次运行（创建向量数据库）

2️⃣ 后续运行（加载已有数据库）

3️⃣ 修改查询问题

常用查询示例：

"崔世亚的教育背景是什么？"

"崔世亚有哪些研究成果？"

"崔世亚的实习经历是什么？"

"崔世亚的研究方向是什么？"

"崔世亚的论文主要研究什么内容？"




💡 常见问题

Q1: 如何重新构建向量数据库？

删除 ./cui_vector_db/ 目录，然后重新运行程序。

Q2: 如何添加新的PDF文件？

将新PDF文件放入 ./知识库文档/ 目录

在代码第157-165行的 pdf_files 列表中添加文件名

删除旧的向量数据库

重新运行程序
